Powerful abbreviation search engine, because apparently googling or asking a colleague for clarification is too time-consuming.

## stack:

- Next.js 13 (with appDir)
- React server component
- Next auth
- Prisma
- ChatGPT API
- Tailwind CSS
- Vercel OG
- Typescript
- ...
